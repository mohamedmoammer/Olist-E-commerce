# 🛒 Olist E-Commerce Sales Analysis | Power BI & SQL

## 📌 Project Overview

This project analyzes the Olist E-Commerce dataset to uncover sales performance, customer behavior, product trends, and operational insights. The analysis was performed using SQL for data extraction and transformation, followed by Power BI for interactive dashboard development and business visualization.

The goal of this project is to help stakeholders understand business performance and make data-driven decisions through interactive reporting.

---

## 🎯 Business Objectives

- Analyze overall sales performance.
- Monitor revenue trends over time.
- Evaluate customer purchasing behavior.
- Identify top-performing product categories.
- Analyze seller performance.
- Track order delivery performance.
- Measure customer satisfaction.
- Support business decisions using data visualization.
